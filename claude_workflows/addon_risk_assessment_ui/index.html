Addon Risk Assessment Ui

Framework: BDD Generated: 2026-06-23 Model: claude-sonnet-4-6
Test Name: Risk Assessment renders and JSON is complete for QA Manager
Feature: Risk Assessment UI
User Role: QA Manager
Traceability: @SHIFT-29@SHIFT-30
Description: Verifies that a QA Manager can run the Risk Assessment add-on on a BDD test file and receive complete, well-structured risk output — both in the on-screen panel and as a downloadable JSON report.
This Test Verifies:
Verification StepWhat It Checks
Documentation and analysis appears in the output panel The Analyze page displays results after processing the uploaded test file with the Risk Assessment add-on
Scenario Risk Ratings appears exactly once in the output The risk rating section for individual scenarios is present and not duplicated in the displayed output
Risk Summary appears exactly once in the output An overall risk summary section is present and not duplicated in the displayed output
Downloaded JSON has no unresolved values The exported report contains no placeholder or missing data fields
JSON contains top-level key 'risk_assessment' The exported report follows the expected structure with a dedicated risk assessment section
'risk_assessment' section has a non-empty 'scenario_risk_ratings' The exported report includes actual risk ratings for the uploaded scenarios, not an empty result
'scenario_risk_ratings' appears exactly once in the 'risk_assessment' section The JSON structure is clean with no duplicate risk rating entries
